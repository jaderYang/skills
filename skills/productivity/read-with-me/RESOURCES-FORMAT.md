# RESOURCES-FORMAT.md

读书/学习方法论的资源清单。根层 `~/reading/RESOURCES.md` 使用此格式。

对标 teach 的 RESOURCES:本 skill 的「不信任脑内知识」原则要求所有背景补充、事实陈述优先查这些已查证源,给出处。

## 资源条目格式

```markdown
### [资源名称]
- **类型**:[书籍 / 论文 / 文章 / 视频 / 课程]
- **作者**:[作者]
- **链接**:[URL 或本地路径]
- **核心要点**:[3-5 条核心主张]
- **适合**:[何时引用]
- **查证状态**:[已查证 / 待查证]
- **查证日期**:YYYY-MM-DD
```

## 初始预填(三条已查证源)

onboarding 创建 `~/reading/RESOURCES.md` 时,预填以下三条(都已在 2026-06-13 查证):

### How to Read a Book(Mortimer Adler)
- **类型**:书籍
- **作者**:Mortimer J. Adler & Charles Van Doren
- **链接**:https://fs.blog/how-to-read-a-book/
- **核心要点**:
  - 四层阅读法:Elementary / Inspectional / Analytical / Syntopical
  - 「苛求读者四问」:谈什么 / 怎么说 / 说得对吗 / so what
  - 检视阅读(快速扫)判断一本书是否值得深读
  - 分析阅读是真正的工作,要主动整合概念
  - 主题阅读是多本书对照建立自己的术语与问题
- **适合**:陪读循环的三模式、每章四问骨架、开新书检视
- **查证状态**:已查证
- **查证日期**:2026-06-13

### How To Take Smart Notes(Sönke Ahrens)/ Zettelkasten
- **类型**:书籍
- **作者**:Sönke Ahrens(基于 Niklas Luhmann 方法)
- **链接**:https://fortelabs.com/blog/how-to-take-smart-notes/
- **核心要点**:
  - 三类笔记:Fleeting / Literature / Permanent
  - 文献笔记必须「用自己的话 + 出处」,不照抄
  - 永久笔记用三问加工:反驳/支持/补充已知?组合出新东西?触发新问题?
  - 按情境(将在哪用到)而非主题组织
  - 刻意保存矛盾观点,对抗确认偏误
  - 标准化格式让规模增长时不失控
  - 写作不是思考的输出,是思考发生的介质
- **适合**:NOTES.md 三区结构、永久笔记加工、跨书链接
- **查证状态**:已查证
- **查证日期**:2026-06-13

### 循证学习法(Feynman + 主动回忆 + 间隔重复)
- **类型**:综合
- **作者**:Richard Feynman(费曼技巧)/ Dunlosky et al.(循证学习策略)
- **链接**:
  - https://www.reddit.com/r/GetStudying/ (社区讨论汇总)
  - Dunlosky et al. 2013 "Improving Students' Learning With Effective Learning Techniques"
- **核心要点**:
  - 费曼技巧:用大白话讲给外行听,讲不清 = 没懂
  - 主动回忆(active recall)比被动重读有效得多
  - 间隔重复(spaced repetition)利用遗忘曲线建长期记忆
  - 测试效应(testing effect):自测本身就是学习
  - 交错练习(interleaving)适合技能类,不适合概念类
- **适合**:复述招式、卡片复习机制、抽考正确率追踪
- **查证状态**:已查证
- **查证日期**:2026-06-13

## 维护规则

- 用户在陪读中发现新的好资源时,追加到本文件(查证状态先标「待查证」,下次验证后改)
- 不删除,只追加
- 任何 skill 引用的资源都应在本文件有登记
